osu-to-CK converter

This tool can be used to chart Caldera's Keep songs easier in osu!mania. It is not designed for application outside of development for Tiny Horse Games' upcoming indie title.
The CK development team currently does not have an accessible way to make our RhythmPlay charts. This fork allows us to use an existing osu! parser as our RhythmPlay minigame format shares a relatively similar structure as osu!mania.

Instructions
1. Import an .mp3 into osu! and setup the map for osu!mania in 6K.
2. Map the left 3 lanes as light notes and the right 3 lanes as dark notes.
3. After mapping the song save and open the .osu beatmap located in the osu!\Songs folder
4. Copy and paste the contents of the .osu beatmap into the converter
5. Adjust any options you need to change and convert
6. Copy the result into the JSON file in CK which is located in Assets/Resources/MusicMinigameCharts/.
7. Test the map in CK


Song ID: (defaults to song name in osu file or "songName" if none)
BPM: (defaults to the highest BPM in osu! file)
Time signature: (defaults to signature of the first timing point)
